# 키 이벤트
## event.preventDefault();
### 현재 기본 이벤트 중단
예)  
```
<a href="https://google.com" id="google">
$("#google").on('click', function(e){
    e.preventDefault();
    // href로 이동 되지 않음
});
```
## 키 이벤트 작동 시점
keypress, keydown, keyup  
키 누르는 시점에 따라 작동  
먼저 실행되는 기능일 수록 키누르는시점이 빠른 것으로 해야함  
> 예)  
> 엔터 검색  
> 알럿창 뜸  
> 엔터 알럿창 닫힘  
> 검색이 알럿창 닫히는 이벤트보다 빠른 시점에 일어나야함  
> 검색: keydown, 알럿 닫힘: keyup
